1️⃣ 🎮 Game Feel (نرفعه لمستوى أعلى)

هذا أهم شيء الآن

🔜 أشياء ناقصة:
✨ Particles (شرارات، دخان)
💥 Debris (حطام الطائرة)
🌟 Glow effect (إضاءة خفيفة)
🔊 Sounds (مهم جدًا جدًا)
2️⃣ 🧠 Game Systems (منطق اللعبة)

اللعبة تعمل، لكن ليست “نظام لعبة كامل” بعد

🔜 تحتاج:
نظام waves (تدرج صعوبة)
نظام score احترافي
نظام lives / respawn
balancing (سرعة + spawn)
3️⃣ 🧩 UI / UX

حاليًا عندك نص بسيط فقط

🔜 نطور:
❤️ شريط حياة (health bar)
🎯 UI للسكور احترافي
💀 شاشة Game Over جميلة
▶️ زر Restart
4️⃣ ⚡ Polish (اللمسات النهائية)

هنا تتحول لعبتك من “جيدة” إلى “ممتازة”

🔜 مثل:
Screen shake متطور (مش بس عشوائي)
Slow motion عند الانفجار 🔥
Hit pause (freeze frame لحظة الاصطدام)
Camera lag (تأخير خفيف بالكاميرا)
5️⃣ 📱 لاحقًا (مو الآن)
دعم الموبايل
touch controls
